# Logo PanicDrive

Metti qui il logo definitivo.

Formato consigliato:

- Simbolo/app icon attualmente usato: `panicdrive-mark.png`.
- Se bitmap: almeno `1024 x 1024 px`, sfondo trasparente.
- Deve essere leggibile anche a `40 x 40 px`.
- Evita testo piccolo dentro al simbolo: il nome PanicDrive e gia scritto in header.

Se cambi nome o estensione, aggiorna `index.html` e `manifest.webmanifest`.
