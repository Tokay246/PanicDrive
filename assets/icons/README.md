# Icone moduli PanicDrive

Metti qui le icone definitive dei moduli.

Formato consigliato:

- `SVG` se vuoi massima nitidezza e peso minimo.
- `PNG` o `WebP` se sono illustrazioni bitmap.
- Canvas quadrato: `512 x 512 px`.
- Area utile centrale: lascia almeno `64 px` di margine su ogni lato.
- Sfondo trasparente.
- Leggibili a `48 px`, senza dettagli troppo fini.
- Peso ideale: sotto `80 KB` per file.

Nomi attualmente usati nello scaffold:

- `exam.png`
- `shield.png`
- `phonebook.png`
- `plus.svg`

Se cambi estensione o nome, aggiorna il campo `icon` in `data/modules.json`.
