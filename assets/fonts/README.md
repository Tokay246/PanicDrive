# Font logo PanicDrive

Per usare il font "Velocity" nel logo:

1. Metti qui il file del font.
2. Rinominalo in uno di questi modi:
   - `velocity.woff2` consigliato
   - `velocity.woff`
   - `velocity.ttf`
3. Ricarica l'app.

Il CSS usa gia `PanicVelocity` come primo font. Se il file non c'e, il browser usa un fallback stile arcade.

Nota: usa un font con licenza compatibile con pubblicazione web/GitHub Pages.
